# TruyenQQ Extension for Paperback

Đọc truyện từ [TruyenQQ](https://truyenqqno.com) trên Paperback app.

## Tính năng

✅ Tìm kiếm truyện  
✅ Browse catalog (Mới cập nhật)  
✅ Đọc truyện Manhwa/Manga/Manhua  
✅ Hỗ trợ tiếng Việt  
✅ Pagination support  

## Cài đặt

### Trên Paperback app (iOS):

1. Mở Paperback app
2. Settings → Sources → External Sources
3. Thêm repository URL:
   ```
   https://raw.githubusercontent.com/YOUR_USERNAME/REPO_NAME/main/
   ```
4. Refresh và install "TruyenQQ"
